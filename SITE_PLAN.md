# 크잉에듀 콘텐츠 제작 앱 기획

## 현재 방향

크잉에듀 사이트는 승인된 원장님이 비밀번호로 접속해 블로그, 썸네일, 릴스 기획을 빠르게 제작하는 콘텐츠 스튜디오입니다.

## 핵심 기능

1. 앱 전체 비밀번호 게이트
   - `/login.html`에서 앱 비밀번호 입력
   - 로그인 후에만 메인 앱 사용
   - OpenAI 생성 API도 로그인 쿠키가 있어야 호출 가능

2. 블로그 템플릿 복제형 글쓰기
   - 원장님 또는 본사가 가진 블로그 템플릿 업로드
   - 템플릿의 구조, 소제목 흐름, 설득 순서를 분석
   - 지역 키워드, 주제, 목적성, 학원 상황에 맞게 새 글로 재작성
   - 네이버 블로그 최적화 규칙 내장

3. 블로그 썸네일 제작
   - 학원 로고 업로드
   - 활동사진 또는 배경이미지 업로드
   - 정사각형 썸네일 생성 및 PNG 다운로드

4. 인스타그램 릴스 기획
   - 릴스 소재, 타깃, 길이 입력
   - 후킹 문장, 화면 구성, 대본, 캡션, CTA 생성

5. 홍보 보조 문구
   - 지역 키워드 예시
   - 학부모 상담 문구
   - 월간 콘텐츠 캘린더
   - 상담 전환 체크리스트

## 운영 메모

- 앱 비밀번호는 Vercel 환경변수 `APP_PASSWORD`로 관리합니다.
- OpenAI API 키는 Vercel 환경변수 `OPENAI_API_KEY`로 관리합니다.
- 템플릿 업로드는 브라우저에서 텍스트를 읽어 AI 요청에 포함하는 방식입니다.
