# 크잉에듀 사이트 기획안

## 목표

크잉에듀 사이트는 영어 교구 수업 운영과 지역 마케팅 실행을 연결하는 원장님 전용 허브입니다. 원장님은 수업 활동을 기록하고, 본사는 활동 피드백과 우수 사례를 모으며, 각 원은 ChatGPT 기반 콘텐츠 생성 도구로 블로그와 인스타그램 홍보물을 빠르게 제작합니다.

## 핵심 사용자

- 원장님: 수업 활동 업로드, 피드백 확인, 지역 홍보 콘텐츠 제작
- 본사 운영자: 활동 품질 확인, 우수 사례 선정, 캠페인 주제 배포
- 마케팅 담당자: 지역 키워드, 릴스 기획, 학부모 상담 문구 관리

## 주요 기능

1. 활동 피드백 보드
   - 원명, 연령, 교구명, 수업 주제, 아이 반응, 사진/영상 업로드
   - 본사 코멘트, 우수 활동 표시, 원장님 간 댓글

2. ChatGPT 포스팅 생성기
   - 입력값: 지역 키워드, 주제, 목적성, 채널, 톤
   - 출력값: 블로그 제목, 본문 구조, 인스타그램 캡션, 해시태그, 상담 CTA
   - 향후 API 연결: 서버에서 OpenAI API 키를 보관하고 프롬프트 템플릿을 호출

3. 릴스 기획 도구
   - 후킹 문장, 장면 구성, 대본, 자막, 캡션, CTA 생성
   - 15초/30초/45초 길이별 템플릿 제공

4. 홍보 키트
   - 지역 키워드 뱅크
   - 학부모 상담 문자
   - 월간 콘텐츠 캘린더
   - 성과 리포트

5. 관리자 기능
   - 캠페인 주제 등록
   - 원별 활동 현황 확인
   - 우수 콘텐츠 템플릿 저장
   - 브랜드 문구와 금칙어 관리

## MVP 범위

- 정적 프로토타입: `index.html`, `styles.css`, `app.js`
- 투명 배경 로고: `assets/kringedu-logo-transparent.png`
- 브라우저 저장 기반 활동 피드
- 입력값 기반 포스팅/릴스/홍보 샘플 생성
