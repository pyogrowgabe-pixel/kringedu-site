# 크잉에듀 원장님 전용관 설계

## 현재 구현

`/members/`에 원장님 전용관 화면을 추가했습니다.

- 전용관 로그인 화면
- 교안 자료 등록 UI
- 등록 자료 목록
- 다운로드 버튼 없는 열람 전용 자료 목록
- 보안 운영 안내

현재 버전은 실제 파일을 서버에 업로드하지 않습니다. 민감한 교안이 공개 저장소나 공개 스토리지에 올라가는 것을 막기 위해, 보안 저장소 연결 전까지는 브라우저에서 등록 흐름만 확인하도록 설계했습니다.

## 실제 운영을 위한 다음 단계

실제 교안 공유 기능을 운영하려면 아래 구성이 필요합니다.

1. Supabase Auth
   - 원장님 계정 생성
   - 승인된 계정만 로그인 허용
   - 본사 관리자 계정 분리

2. Supabase Storage
   - 교안 파일 비공개 버킷 저장
   - 자료별 접근 권한 관리
   - 공개 URL 대신 짧은 만료 시간의 signed URL 사용

3. 자료 메타데이터 DB
   - 자료명
   - 분류
   - 대상 연령
   - 업로드일
   - 업로드한 관리자
   - 열람 권한

4. 보안 보완
   - 원장명/학원명 워터마크
   - 열람 로그 저장
   - 다운로드 가능 자료와 열람 전용 자료 구분
   - 중요 교안은 일부 미리보기만 제공

## 중요한 한계

웹에서 사용자가 볼 수 있는 자료는 완벽한 복사 차단이 불가능합니다. 보안 목표는 “절대 복사 불가”가 아니라 “권한 없는 접근을 막고, 유출 가능성을 낮추고, 유출 시 추적할 수 있게 만드는 것”입니다.
