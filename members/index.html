Members Library

원장님 전용 자료실

교안, 월간 수업 자료, 마케팅 자료를 승인된 원장님 전용으로 관리하는 공간입니다.

보안 원칙 자료 공유는 로그인, 워터마크, 열람 기록, 권한 설정을 기준으로 운영합니다.

Access

전용관 로그인

현재 버전은 전용관 화면과 업로드 흐름을 확인하는 단계입니다. 실제 공유형 업로드는 보안 저장소 연결 후 활성화합니다.

Upload

교안 자료 등록

실제 파일은 아직 서버에 저장하지 않고, 이 브라우저에서 자료 등록 흐름만 확인합니다.

Private Materials

등록 자료

다운로드 차단보다 중요한 것

웹에서 보이는 자료는 완전한 복사 차단이 불가능합니다. 대신 로그인, 권한, 워터마크, 열람 기록으로 유출 가능성을 낮추는 방식이 안전합니다.

다음 연결 단계

Supabase Auth와 Storage를 연결하면 승인된 원장님 계정만 자료를 열람하고, 자료별 권한과 워터마크를 적용할 수 있습니다.

권장 운영

핵심 교안은 다운로드 대신 뷰어 중심으로 제공하고, 원장명/학원명 워터마크를 자동 삽입하는 방식을 권장합니다.
